# License Plate Detection and Recognition

This project provides a solution for detecting and recognizing license plates from images using MATLAB. The implementation includes scripts for plate detection, character recognition, and template creation for character matching.

## Introduction
The License Plate Detection and Recognition project aims to automate the process of identifying license plates in images and recognizing the characters on those plates. This is useful for various applications such as automatic toll collection, parking management, and traffic monitoring.

## Features
- **Plate Detection:** Detects the location of the license plate in an image.
- **Character Recognition:** Identifies individual characters from the detected license plate.
- **Template Creation:** Creates templates for each character for matching and recognition.

## Installation
To use this project, you need to have MATLAB installed on your system. Follow the steps below to set up the project:

1. Clone the repository:
   ```bash
   git clone https://github.com/ahnaf-tahmid-zaman/license-plate-detection.git
2. Navigate to the project directory:

    ```bash
    cd Number Plate Detection In Matlab
    ```
3. Open MATLAB and add the project directory to your MATLAB path:

    ```bash
    addpath('path_to_project_directory')
    ```
## Usage
Run the `Plate_detection.m` script to detect the license plate in an image.
